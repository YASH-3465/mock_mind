# Mock Mind

AI-Powered Interview Preparation Platform

## Overview

Mock Mind is an AI-powered platform that helps students and professionals prepare for technical interviews through personalized mock interviews, resume analysis, and AI-generated feedback.

---

## Tech Stack

### Backend
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic

### Authentication
- JWT
- Passlib (bcrypt)

### Frontend
- React (Coming Soon)

### AI
- Whisper
- LLM (Coming Soon)
